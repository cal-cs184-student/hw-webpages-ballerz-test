Section I: Bezier Curves and Surfaces

Part 1: Bezier curves with 1D de Casteljau subdivision

De Casteljau’s Algorithm and Implementation

    In my implementation, I iterated through the original control points, then applied the interpolation formula (pi’ = (1-t)*points[i] + t*points[i+1]) to each pair of consecutive points. I stored all of these newly computed points in the vector intermediate_control_pts, which represents the intermediate control points at the parameter t in the next level of subdivision.

Part 2: Bezier surfaces with separable 1D de Casteljau

Bezier Surface Implementation

    De Casteljau’s algorithm extends to Bezier surfaces when it is first evaluated with the parameter u, then with the parameter v. To implement this, I evaluated each of the Bezier curves with the parameter u. This gave me a set of intermediate points, which I treated as the new set of control points and evaluated with the parameter v, to calculate the evaluated point on the Bezier surface.

Section II: Triangle Meshes and Half-Edge Data Structure

Part 3: Area-weighted vertex normals

Implementation

    To implement the area-weighted vertex normals, I used the half-edge data structure to iterate through all faces that are adjacent to the vertex. For each face, I calculated the two edge vectors, from the current position to the two neighboring vertices. Then, I calculated the cross product of these two edge vectors and added it to the total. At the end, I normalized this total to get the area-weighted vertex normal.
